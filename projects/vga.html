VGA Display Controller

SystemVerilog DE1-SoC

Graphics rendering pipeline driving a 160×120 VGA framebuffer at one pixel per clock cycle.

Overview

This project builds a complete graphics engine that renders geometric shapes to a VGA monitor through the DE1-SoC's video output. The pipeline writes to a 160×120 pixel framebuffer at a throughput of one pixel per clock cycle. It implements Bresenham's circle algorithm using pure integer arithmetic, an 8-fold symmetry optimization via an octant-rotating counter, and Reuleaux triangle construction using fixed-point √3 arithmetic with 10-bit scaling. All drawing includes combinational bounds checking to clip pixels outside the visible area.

What I Built

  • fillscreen — Fills the entire framebuffer with a colour pattern, iterating through all 19,200 pixels sequentially
  • circle — Draws circles using Bresenham's midpoint algorithm; a 3-bit octant counter plots 8 symmetric points per iteration
  • reuleaux — Constructs Reuleaux triangles from three overlapping partial circle arcs, computing arc centers with fixed-point √3 math

Design Highlights

  • One pixel per clock cycle — The pipeline architecture writes a pixel every clock cycle during active drawing, maximizing framebuffer throughput without stalls or wait states.
  • 8-fold symmetry via octant counter — Instead of computing 8 separate pixels, a single (offset_x, offset_y) pair is reflected across all octants using a 3-bit rotating counter. This produces a complete circle in ⅛ the Bresenham iterations.
  • Fixed-point √3 arithmetic — Reuleaux triangle vertices require √3 offsets from the center. A 10-bit fixed-point representation (multiply by 1774, right-shift by 10) avoids floating point entirely while maintaining sub-pixel accuracy at the 160×120 resolution.

Code: Bresenham Circle — Octant Drawing

Each Bresenham step computes one (offset_x, offset_y) pair. The 3-bit ptest counter reflects it across all 8 octants before advancing to the next midpoint decision.

// circle.sv — 8-fold symmetric Bresenham circle drawing

if (offset_y <= offset_x) begin
    color <= colour;
    plot  <= 1'b1;

    if (ptest == 3'b000) begin        // Octant 1
         x <= centre_x + offset_x;
         y <= centre_y + offset_y;
         ptest <= ptest + 1'b1;
    end else if (ptest == 3'b001) begin  // Octant 2
         x <= centre_x + offset_y;
         y <= centre_y + offset_x;
         ptest <= ptest + 1'b1;
    end else if (ptest == 3'b010) begin  // Octant 3
         x <= centre_x - offset_y;
         y <= centre_y + offset_x;
         ptest <= ptest + 1'b1;
    end else if (ptest == 3'b011) begin  // Octant 4
         x <= centre_x - offset_x;
         y <= centre_y + offset_y;
         ptest <= ptest + 1'b1;
    end else if (ptest == 3'b100) begin  // Octant 5
         x <= centre_x - offset_x;
         y <= centre_y - offset_y;
         ptest <= ptest + 1'b1;
    end else if (ptest == 3'b101) begin  // Octant 6
         x <= centre_x - offset_y;
         y <= centre_y - offset_x;
         ptest <= ptest + 1'b1;
    end else if (ptest == 3'b110) begin  // Octant 7
         x <= centre_x + offset_y;
         y <= centre_y - offset_x;
         ptest <= ptest + 1'b1;
    end else if (ptest == 3'b111) begin  // Octant 8 — advance Bresenham
         x <= centre_x + offset_x;
         y <= centre_y - offset_y;
         ptest <= 3'b000;

         offset_y <= offset_y + 10'sd1;
         if (crit <= 8'sd0)
              crit <= crit + 2 * (offset_y + 1) + 1;
         else begin
              offset_x <= offset_x - 10'sd1;
              crit <= crit + 2 * ((offset_y + 1) - (offset_x - 1)) + 1;
         end
    end
end

Code: Bounds Clipping

Pixels outside the 160×120 framebuffer are suppressed combinationally — the VGA plot signal is gated so no write occurs for out-of-bounds coordinates.

// circle.sv — combinational clipping logic

always_comb begin
    if (STATE != 1'b0)
        if ((x >= 10'sd0) && (y >= 10'sd0) &&
            (x < 10'sd160) && (y < 10'sd120))
            in_bounds = 1'b1;
        else
            in_bounds = 1'b0;
    else
        in_bounds = 1'b1;
end

assign vga_plot = plot & in_bounds;
